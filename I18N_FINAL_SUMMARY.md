## 🌐 语言切换功能

### 位置
**Settings → General → Appearance → Language**

### 支持的语言
- 🌐 **浏览器默认** (BROWSER)
- 🇬🇧 **英语** (EN)
- 🇫🇷 **法语** (FR)
- 🇯🇵 **日语** (JA)
- 🇪🇸 **西班牙语** (ES)
- 🇰🇷 **韩语** (KO)
- 🇨🇳 **简体中文** (ZH_HANS) ✨
- 🇹🇨 **繁体中文** (ZH_HANT)

### 使用方法
1. 打开应用
2. 点击右上角设置（⚙️）
3. 在"常规"页面找到"语言"
4. 选择"简体中文"
5. 点击"保存设置"按钮
6. **界面立即切换为中文** ✨
